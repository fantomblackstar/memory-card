# Memory Card Game

A fun and interactive memory card matching game built with HTML, CSS, and JavaScript.

**[Play the game here](https://fantomblackstar.github.io/memory-card/)**

## How to Play

- Click on cards to reveal them.
- Try to find matching pairs.
- The game ends when all pairs are found.
- Time your moves and test your memory!

## Features

- Simple and intuitive card-flip mechanics.
- Responsive design that works on desktop and mobile.
- Score tracking.
- Built with vanilla JavaScript (no frameworks).

## Technologies

- HTML5
- CSS3
- JavaScript
